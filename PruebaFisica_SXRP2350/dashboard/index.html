🎛️ Control Manual (Demo)

TEMP:65 EXT:25 SET:50 TEMP:AUTO EXT:AUTO SET:AUTO

SET:<°C> fija setpoint · TEMP:<°C> fuerza temp. interna · EXT:<°C> fuerza temp. externa · usa AUTO para volver al sensor NTC.

⚠️ Requiere Chrome/Edge de escritorio (Web Serial API). Firmware a 115200 baudios.

Al pulsar Conectar, selecciona el puerto COM del XIAO RP2350. El dashboard parsea la línea TLM … emitida cada segundo por el firmware.

🧰 Tecnologías utilizadas

🖥️ Frontend / Dashboard

  • HTML5 + CSS3Glassmorphism, grid responsivo, animaciones CSS (aspas, aurora)
  • JavaScript (Vanilla ES6+)Sin frameworks ni build; un solo archivo autónomo
  • Web Serial APILectura/escritura del puerto COM a 115200 bd desde el navegador
  • Tipografía Inter / JetBrains MonoUI moderna + números monoespaciados

🔌 Firmware / Embebido

  • Seeed XIAO RP2350MCU dual-core; ADC 12 bits para NTC internos/externos
  • C++ (Arduino / PlatformIO)FSM no bloqueante a 1 Hz, watchdog HW, filtro trimmed-mean
  • TB6612FNG + FIT0803Puente H para actuador de compuertas
  • MOSFET NMOS + MR1238E48BConmutación del ventilador de 48 V

🌐 IoT / Backend

  • Python 3Puente Serial → MQTT con perfiles multi-extractor
  • pyserialLectura del puerto serial con reintentos
  • paho-mqttPublicación MQTT con reconexión automática y LWT
  • MQTT + JSONPayload industrial en jaguar/telemetry/data
