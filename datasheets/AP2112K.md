# AP2112K-3.3 — Regulador LDO 3.3V (Diodes Incorporated)

**Función en el sistema:** Regulador de 3.3V para alimentación del MCU y lógica (TB6612FNG VCC, pull-ups, etc.)

---

## Parámetros eléctricos clave

| Parámetro | Valor | Nota |
|---|---|---|
| Tensión de salida | **3.3V** (fija) | Versión K-3.3 |
| Tensión de entrada máx | **6.0 V** | Típico: 5V USB o regulado |
| Tensión de entrada mín | ~3.5V | VIN > VOUT + Vdropout |
| Corriente de salida máx | **600 mA** | |
| Dropout voltage | 250 mV típico | @ 600 mA, podría ser más con margen térmico |
| PSRR | 65 dB @ 1kHz | Muy buena para aplicaciones analógicas |
| Precisión de tensión | ±1.5% | |
| Ruido de salida | 50 µVrms | 10Hz–100kHz (importante para ADC) |
| Temperatura de operación | -40°C a +125°C | |
| Encapsulado | SOT-25 (3-pin) | SMD compacto |
| Enable pin | Pin EN (active HIGH) | Puede conectarse a VIN para always-on |

---

## Esquema de conexión típico

```
VIN (5V) ──┬── AP2112K-3.3 (VIN) ──(regula)── VOUT (3.3V) ──→ MCU, TB6612FNG VCC, lógica
           │          │
       [100nF]    [EN → VIN]
           │          │
          GND      [1µF C_OUT]
                       │
                      GND
```

**Condensadores recomendados:**
- CIN: 1 µF cerámico (cerca de VIN)
- COUT: 1 µF cerámico (cerca de VOUT)
- Opcional: 10 µF electrolítico en VOUT para estabilidad con cargas dinámicas

---

## Consumo de corriente en este proyecto

| Carga | Corriente aprox |
|---|---|
| RP2040 / RP2350 | ~20–100 mA (según actividad) |
| TB6612FNG (VCC lógica) | ~3–5 mA |
| Pull-ups (×5 GPIOs) | < 2 mA |
| **Total estimado** | **~30–110 mA** |
| Margen AP2112K | 600 mA − 110 mA = **490 mA libre** ✓ |

---

## Nota de diseño PCB

- Colocar el AP2112K entre el conector de alimentación y el MCU
- El plano de GND bajo el LDO disipa calor: 0.6A × 0.25V = 150 mW → ΔT < 5°C en SOT-25 sin disipador
- No requiere capacitor de tantalio, cerámica suficiente
